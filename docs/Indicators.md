# Blueprint Indicators

Each year Blueprint indicators are updated within a spreadsheet containing
their names, descriptions, and links to the SECAS Hub site. This spreadsheet
is prepared for the Blueprint Explorer and used here.

Download that spreadsheet to `source_data/Indicators_<year>.xlsx`.

The essential information from this spreadsheet is extracted using
`analysis/prep/extract_indicators.py`.

This outputs a file to `src/routes/blueprint-indicators/indicators.json`.

See [analysis/prep/README.md](../analysis/prep/README.md) for more information.
